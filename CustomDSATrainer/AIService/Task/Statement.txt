Title: Simple Addition

Statement:
You are given two integers, a and b. Calculate and output their sum.

Constraints:
0 <= a, b <= 1000

Example 1:
Input:
1
2
Output:
3
Explanation:
1 + 2 = 3

Example 2:
Input:
10
20
Output:
30
Explanation:
10 + 20 = 30

Example 3:
Input:
0
0
Output:
0
Explanation:
0 + 0 = 0
